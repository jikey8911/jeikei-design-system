# JeiKei Design Principles 🧠

## 1. Futurismo Funcional
El sistema debe sentirse como una interfaz avanzada, inspirada en controles de misión y redes neuronales. La estética no debe comprometer la usabilidad.

## 2. Neural Connection
La interfaz debe estar viva. El fondo neuronal no es estático, reacciona a los pulsos de los componentes y a la interacción del usuario.

## 3. Glass & Glow
Uso de efectos de vidrio esmerilado (`backdrop-blur`) para profundidad y resplandores neón para jerarquía visual y retroalimentación de estado.

## 4. Accesibilidad Cibernética
Aunque el estilo es oscuro y futurista, debemos mantener contrastes legibles y soporte completo para navegación por teclado (ARIA).

## 5. Escala Atómica
Desde tokens hasta dashboards complejos, cada pieza del sistema debe ser modular y reutilizable.
